Fórmulas de indicadores

Cálculos aplicados por fn_indicadores_pollo_engorde

Población
Mortalidad %
mortalidad / aves_encasetadas × 100
Supervivencia %
( aves_encasetadas − mortalidad ) / aves_encasetadas × 100
Ajuste de aves
aves_encasetadas − aves_vendidas − ( mortalidad + selección )
% Ajuste
ajuste_aves / aves_encasetadas × 100
Merma % solo si merma
merma_unidades / aves_vendidas × 100
Alimento
Consumo ave (g)
( consumo_total_kg / aves_vendidas ) × 1000
Tiempo
Días de engorde
fecha_cierre − fecha_encasetamiento
Producción
Kg carne
Σ peso_neto por movimiento de despacho ¹
Peso promedio (kg)
kg_carne / aves_vendidas
Producción kilo en pie
= kg_carne
Total kilos a cliente solo si merma
producción_kilo_en_pie − merma_kilos
Eficiencias
Conversión
consumo_total_kg / kg_carne
Conversión ajustada P=2.7 · D=4.5
conversión + ( ( P − peso_prom ) / D )
Eficiencia americana
( peso_prom / conversión ) × 100
Eficiencia europea (IEEP)
( peso_prom × supervivencia% ) / ( días × conversión ) × 100
Índice de productividad
( peso_prom / conversión² ) × 100
Ganancia / día (g)
( peso_prom / días_engorde ) × 1000
Densidad
Aves / m²
aves_vendidas / m²
Kg / m²
kg_carne / m²
¹ Si no hay peso_neto: usa peso_bruto − peso_tara. Fuente: movimiento_pollo_engorde tipo Venta / Despacho / Retiro.
mortalidad incluye selección del seguimiento diario: mortalidad_hembras + mortalidad_machos + sel_h + sel_m.
P / D son parámetros configurables por lote (por defecto 2.7 / 4.5).
