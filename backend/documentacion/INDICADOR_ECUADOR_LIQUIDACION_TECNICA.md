# Indicador Ecuador – Liquidación Técnica

Existen **dos** liquidaciones técnicas distintas:

1. **Lotes Levante/Producción** (tablas `lotes`, `movimiento_aves`, seguimientos levante/producción): es la que existía anteriormente para el indicador general.
2. **Pollo Engorde** (Lote Ave Engorde y Lote Reproductora Ave Engorde): liquidación específica para Ecuador con tablas de pollo engorde.

---

Para la **Liquidación Técnica de Pollo Engorde** (lote padre y lotes reproductores), indicadores, fórmulas y reglas de liquidación de granjas, ver el documento dedicado:

→ **[LIQUIDACION_TECNICA_POLLO_ENGORDE.md](./LIQUIDACION_TECNICA_POLLO_ENGORDE.md)**

Ese documento es el de referencia **solo para pollo engorde** y es independiente de la liquidación anterior (Levante/Producción).
